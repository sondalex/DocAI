## Development

Run unit tests:

```bash
bun test
```

Development server:

```bash
bun run dev
```

## Deployment

Set up your baseurl using `VITE_BASE_URL` environment variable, run 

```bash
bun run build
```

And then push `dist/` folder to your server.
